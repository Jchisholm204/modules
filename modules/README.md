# Modules

Place software builds here.

## Folder Format
```
modules/
├── module_name/
│   └── version/
│       ├── bin/
│       ├── doc/
│       ├── share/
│       └── lib/
└── cmake/
    ├── 3.30.2/
    │   ├── bin/
    │   ├── doc/
    │   └── share/
    └── 3.26.4/
        ├── bin/
        ├── doc/
        └── share/
```
